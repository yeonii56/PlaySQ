## 👉 프로젝트 소개
- 게임 장르 : 물리기반 2D 퍼즐
- 플랫폼 : 윈도우
- 제작 인원 : 4 인 (기획1, 디자인1, 개발1, 이펙터1)
- 제작 기간 : 각 2주 + QA 1주 (총 9개월)
- 사용 엔진과 언어 : Unity, C#
- 작업물: 2인용 9개, 다인용 2개
